# H6629 - tsón

**Definición:**
no tseón (Sal_144:13); de una raíz que no se usa que significa emigrar; nombre colect. de un rebaño (de ovejas o cabras); también figurativamente (de hombres):- borrego, corderito, cordero, ganado, grey, hato, majada, manada, novillo, oveja, ovejuno, pastar, pastor, rebaño, redil.
