# H376 - ish

**Definición:**
ontracción de [[H582]] [o tal vez más de una raíz que no se usa que significa ser existente]; hombre como individuo o persona masculinos; que frecuentemente se usa como adjunto para un término más def. (y en tales casos frecuentemente no se expresa en las traducciones):- alguno, cadáver, casado, digno, extranjero, extraño, hombre, humano, Ishi, labrador, macho, marido, marinero, paladín, persona, soldado, varón. Compárese con [[H802]].
