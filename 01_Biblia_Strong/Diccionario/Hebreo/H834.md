# H834 - asher

**Definición:**
ronombre relativamente primario (de todo género y número); quién, cuál, qué, que; también (como adverbio y como conjunción) cuándo, dónde, cómo, porque, para que, etc.:- cuándo, en lugar, cuándo, dónde, cómo, porque, para que. [Dado que es indeclinable, frecuentemente lo acompaña el pronombre personal como interjección, usado para mostrar la relación.]
