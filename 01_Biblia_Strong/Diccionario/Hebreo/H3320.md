# H3320 - yatsáb

**Definición:**
braíz primaria; colocar (cualquier cosa como para que se quede); reflexivamente estacionar, ofrecer, continuar:- afirmar, aguardar, dejar, detener, esperar, firme, hallar, juntar, levantar, parar, pie, poner, presentar, presente, resistir, sostener.
