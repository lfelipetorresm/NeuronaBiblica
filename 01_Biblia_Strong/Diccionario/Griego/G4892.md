# G4892 - sunédrion

**Definición:**
drionneutro de un derivado presunto de un compuesto de [[G4862]] y la base de [[G1476]]; sesión conjunta, i.e. (específicamente) el sanedrín judío; por analogía tribunal subordinado:- concilio.
