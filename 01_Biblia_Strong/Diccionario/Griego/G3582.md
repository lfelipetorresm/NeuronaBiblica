# G3582 - xéstes

**Definición:**
stescomo si de xéo (propiamente alisar; por implicación [de fricción] hervir o calentar); vasija (como moldeada o para cocinar) [o tal vez por corrupción del latín sextario, sexto de un modio, i.e. alrededor de una pinta], i.e. (específicamente) medida para líquidos o áridos, (por analogía jarra):- jarros.
