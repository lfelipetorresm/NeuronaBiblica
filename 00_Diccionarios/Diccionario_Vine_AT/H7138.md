# [[H7138]]

qarôb (קָרויב, [[H7138]]), «cerca». Este término aparece unas 77 veces. Qarôb puede referirse tanto a proximidad espacial (Gén_19:20 : primer caso en la Biblia) como a aproximación epistemológica (Deu_30:14 rva). El adjetivo también aparece en Eze_6:12 : «El que esté lejos morirá de peste, el que esté cerca caerá a espada».

El adjetivo qareb tiene significados paralelos a qarôb. Qareb, que se halla 11 veces, significa «cerca» y se refiere a una proximidad íntima (generalmente en un contexto o actividad cúltica). Un ejemplo está en Eze_45:4 : «Lo consagrado de esta tierra será para los sacerdotes, ministros del santuario, que se acercan para ministrar a Jehová: y servirá de lugar para sus casas, y como recinto sagrado para el santuario».
