# TENER MEMORIA

mimnesko (μιμνήσκω, [[G3403]]), se traduce «para que tengáis memoria» (2Pe_3:2); «tened memoria» (Jud_1:17); véanse MEMORIA, B, Nº 1, RECORDAR, A, Nº 2.
