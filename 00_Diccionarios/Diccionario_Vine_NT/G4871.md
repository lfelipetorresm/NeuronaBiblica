# [[G4871]]

sunalizo (óõíáëßæù, [[G4871]]), reunir juntos, juntar, con la sugerencia de una reunión multitudinaria (sun, con; jalizo, agolpar; el adjetivo correspondiente es jales, repleto de gente). Se usa en Hch_1:4. No se debe aceptar el significado de comer con, sugerido por algunos, como si este término se derivara de jals, «sal».¶
