# TIRAR

1. airo (áἴñù, [[G142]]), levantar, llevar arriba o afuera. Se traduce con el verbo «tirar» en Mat_9:16; Mar_2:21, de la acción de un remiendo nuevo sobre un tejido viejo. (Cf. REMIENDO bajo REMENDAR, B, Nº 3). Véase QUITAR, A, Nº 1, etc.

2. balo (âÜëëù, [[G906]]), arrojar, lanzar. Se traduce con el verbo tirar en Jua_8:59 (rv: «tomaron … piedras para tirarle», rvr: «arrojar … »). Véanse ABONAR, ACOSTAR, ARROJAR, DAR, DEJAR CAER, DERRAMAR, DERRIBAR, ECHAR, IMPONER, LANZAR, METER, PONER, POSTRAR(SE), SEMBRAR, TENDER, TRAER.

3. epanago (ἐðáíÜãù, [[G1877]]), véase BOGAR; se traduce «tira a alta mar» (Luc_5:4, rv). Véanse también APARTAR, VOLVER.
