# [[G5571]]

pseudes (øåõäÞò, [[G5571]]), se usa de testigos falsos (Hch_6:13 : «falsos»); de falsos apóstoles (Apo_2:2; Apo_21:8 : «mentirosos»). Véanse MENTIR, MENTIRA, MENTIROSO.¶

Nota: Para palabras compuestas con este adjetivo, véanse, ( FALSOS ) CRISTOS, HERMANO, MAESTRO, PROFETA, TESTIGO, TESTIMONIO.

pseudes (øåõäÞò, [[G5571]]), mentiroso, falso; cf. el prefijo castellano «pseudo –». Se traduce «falso» en Hch_6:13. Se traduce «mentirosos» en Apo_2:2; Apo_21:8. Véase FALSO.¶

Nota: Se forman muchos nombres compuestos mediante el prefijo pseudo –; p.ej., véanse CRISTOS, HERMANO, MAESTRO, PROFETA, TESTIGO.
