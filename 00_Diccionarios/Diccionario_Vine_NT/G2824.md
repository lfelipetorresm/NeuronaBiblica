# [[G2824]]

klima (κλίμα, [[G2824]]), primariamente una inclinación, una pendiente (cf. los términos castellanos clima, climatología). Se utiliza de una región (Rom_15:23 : «regiones», rv, rvr; 2Co_11:10 : «regiones», rv: «partes»; Gál_1:21 : «regiones», rv: «partes»). Véase PARTE.¶
