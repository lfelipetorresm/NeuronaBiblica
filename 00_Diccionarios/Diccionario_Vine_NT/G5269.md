# [[G5269]]

jupozonnumi (õ̔ðïæù́ííõìé [[G5269]]), (jupo, bajo; zonnumi, ceñir). Se usa de atortorar una nave (Hch_27:17), manteniendo el casco de una nave mediante fuertes cuerdas.
