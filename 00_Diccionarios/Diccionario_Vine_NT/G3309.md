# [[G3309]]

merimnao (μεριμνάω, [[G3309]]), véase AFANAR(SE), B, denota estar ansioso, lleno de cuitas. Se traduce «se interese» en Fil_2:20. 

merimnao (μεριμνάω, [[G3309]]), relacionado con A, Nº 1, significa estar ansioso acerca de, tener un cuidado que perturba (p.ej., Mat_6:25, Mat_6:28, «os afanáis»; Mat_10:19; Luc_10:41; Luc_12:11); tener congoja (1Co_7:32-33, 1Co_7:34); preocuparse (1Co_12:25); interesarse (Fil_2:20); «estar afanoso» (Fil_4:6). Véanse CUIDADO ( TENER ), INTERESAR(SE), PREOCUPAR(SE).

Nota : El verbo merimnao se traduce como adjetivo en Fil_4:6, formando parte de la cláusula verbal «estar afanoso». 

merimnao (μεριμνάω, [[G3309]]), afanarse, estar afanoso. Se traduce «tiene cuidado» en 1Co_7:32-33, 1Co_7:34, dos veces; véanse AFANAR(SE), B, CUIDADO, INTER ESAR(SE), PREOCUPAR(SE), SOLICITUD. 

merimnao (μεριμνάω, [[G3309]]), denota estar ansioso, preocupado, y se traduce «no os preocupéis» (Luc_12:11); 1Co_12:25 : «se preocupen»; véanse AFANAR(SE), B, CUIDADO, B, Notas (3) INTERESAR(SE), B, Nº 2, TENER CUIDADO.
