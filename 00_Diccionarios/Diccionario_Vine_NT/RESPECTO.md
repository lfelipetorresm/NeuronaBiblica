# RESPECTO

Notas: (1) Para meros, traducido «en este respecto» en 2Co_3:10, véase PARTE, Nº 1, etc.; (2) juper es una preposición que, con el caso genitivo, significa «a propósito de», «respecto a», p.ej., Rom_1:8 (tr); Rom_9:27 : «tocante»; 2Co_1:7 : «respecto»; 2Co_7:4 : «con respecto de»; v. 14: «respecto de». Véanse también ACERCA, FAVOR, LUGAR, MAS, SUPERIOR.
