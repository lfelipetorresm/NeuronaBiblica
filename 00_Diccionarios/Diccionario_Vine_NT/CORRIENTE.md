# CORRIENTE

aion (áἰþí, [[G165]]), una edad (véase SIGLO). Se menciona alguna vez, erróneamente, como una «dispensación», que no significa un período de tiempo, sino un modo de tratar. Se traduce «corriente» en Efe_2:2 : «la corriente de este mundo», esto es, el ciclo o curso presente de las cosas. Véanse ETERNIDAD, SIEMPRE, SIGLO, UNIVERSO, etc.
