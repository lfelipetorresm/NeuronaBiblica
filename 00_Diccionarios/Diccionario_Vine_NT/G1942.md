# [[G1942]]

epikalumma (ἐðéêÜëõììá, [[G1942]]), una cubierta, un medio tras el cual esconderse (epi, sobre; kalupto, cubrir); de ahí, pretexto, excusa, para la maldad (1Pe_2:16).¶ En la lxx se utiliza en Éxo_26:14 : «cubiertas»; Éxo_39:21 : «que estuviese sobre»; 2Sa_17:19 : «extendió»; Job_19:29 : «engaño» (gr.).¶
