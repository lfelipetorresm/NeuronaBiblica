# MALIGNO

1. kakos (κακός, [[G2556]]), malo. Se traduce «maligna» en Apo_16:2, de una úlcera, acompañando al término poneros, «pestilente» (Véase Nº 2). Véase MAL, A, Nº 1, etc.

2. poneros (πονηρός, [[G4190]]), véanse MAL, A, Nº 2, B, Nº 6. Se traduce «maligno»: (a) del ojo (Mat_6:23; Luc_11:34); (b) de palabras ( 3Jn_1:10); (c) como nombre propio, de Satanás, el «Maligno» (Efe_6:16; 1Jn_2:13-14; 1Jn_3:12; 1Jn_5:18), véanse también MALO, MALVADO, PEOR, PERVERSO, PESTILENTE.

3. skolios (σκολιός, [[G4646]]), véase DIFÍCIL, Nº 4. Se traduce «una generación maligna» (Fil_2:15). Véase también PERVERSO.
