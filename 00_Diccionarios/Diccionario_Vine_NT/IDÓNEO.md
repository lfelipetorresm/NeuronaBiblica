# IDÓNEO

jikanos (ἱêáíüò, [[G2425]]), primariamente que alcanza a; tiene el sentido de «suficiente». Se traduce «idóneos para enseñar» en 2Ti_2:2; véanse COMPETENTE, DIGNO, FIANZA, GRANDE, MUCHO, SUFICIENTE.
