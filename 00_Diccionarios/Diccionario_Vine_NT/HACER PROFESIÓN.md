# HACER PROFESIÓN

jomologeo (ὁìïëïãÝù, [[G3670]]), se traduce «habiendo hecho» en 1Ti_6:12, junto con el nombre que sigue, jomologia : «profesión»; esto es, «habiendo hecho … profesión» (lit: «habiendo profesado la buena profesión»); véase CONFESAR, A, Nº 1, etc.
