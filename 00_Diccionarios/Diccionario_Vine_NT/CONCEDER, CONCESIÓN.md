# CONCEDER, CONCESIÓN

A. Verbos

1. ago (ἄãù, [[G71]]), conducir, mantener, o pasar un día. Se usa en Hch_19:38 : «audiencias se conceden» (rv: «se hacen»). Véanse LLEVAR, TRAER, etc.

2. didomi (äßäùìé, [[G1325]]), dar, se traduce «conceder» (Mar_10:37 : «concédenos», rv: «danos»; Luc_1:73, rv: «dar»; Hch_4:29 : «concede», rv: «da»; Hch_11:17 : «concedió», rv: «dio»; Hch_14:3 : «concediendo», rv: «dando»; 1Co_3:5, rv, rvr: «concedido»; 2Ti_1:18 : «conceda», rv: «dé»; 2Ti_2:25 : «conceda». rv: «dé»; Apo_19:8 : «concedido», rv: «dado»). Véase DAR, etc.

3. carizomai (÷áñßæïìáé, [[G5483]]), el término significa primariamente mostrar favor o bondad; relacionado con caris, véase GRACIA (Gál_3:18 : «concedió», rv: «la donación»); y de ello, dar libremente (Hch_27:24 : «ha concedido», rv: «ha dado»); significa más que simplemente dar (1Co_2:12 : «ha concedido» (rv: «ha dado»; vm: «han sido dadas gratuitamente»; rvr77: «ha otorgado gratuitamente»; Fil_1:29 : «concedido», rv, rvr; Flm_1:22 : «concedido», rv, rvr). Véanse DAR, ENTREGAR, GRATUITAMENTE, PERDONAR.

4. pareco (ðáñÝ÷ù, [[G3930]]), en la voz activa, significa tener medios, ofrecer, proveer, dar, y se traduce con el verbo conceder en Luc_7:4 (rv, rvr). Véanse ACARREAR, CAUSAR, DAR, GUARDAR, HACER, PRESENTAR, TRATAR.

5. epitrepo (ἐðéôñÝðù, [[G2010]]), lit., volverse a (epi, a, trepo, volverse, girar), confiar. Significa permitir, y se traduce con el verbo conceder en Jua_19:38, del permiso dado por Pilato para quitar el cuerpo del Señor Jesús de la cruz, y sepultarlo (rv: «permitió»). Véanse DEJAR, PERMISO, PERMITIR.

B. Nombre

sungnome (óõããíþìç, [[G4774]]), lit., opinión, mente, o entendimiento común (sun, con; gnome, opinión), sentimiento común de simpatía; de ahí, concesión, permiso. Se traduce «concesión» en contraste a «mandamiento», en 1Co_7:6 (rv: permisión).¶
