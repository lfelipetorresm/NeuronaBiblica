# CONSULTAR

prosanatithemi (ðñïóáíáôßèçìé, [[G4323]]), se traduce «consulté» en Gál_1:16. Véase COMUNICAR, Nº 2.

Nota: Sumboulion (Óõìâïýëéïí, [[G4824]]), palabra del período grecorromano, relacionada con el verbo sumbouleo, tomar consejo juntamente, véase ACONSEJAR, es un nombre que se traduce varias veces como «consejo» (véase ACONSEJAR, B, Nº 4). Se traduce como verbo: «consultar», en varias ocasiones (Mat_22:15 : «consultaron», rvr77: «deliberar»; Mat_27:7 : «consultar», rv: «habido consejo»). Véase CONSEJO, en ACONSEJAR, B.
