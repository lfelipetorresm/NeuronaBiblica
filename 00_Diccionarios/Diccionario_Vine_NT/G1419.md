# [[G1419]]

dusbastaktos (äõóâÜóôáêôïò, [[G1419]]), difícil de llevar (de dus –, prefijo inseparable, como el castellano in–, indicativo de dificultad, de daño, oposición, etc., y bastazo, llevar). Se usa en Luc_11:46, y, en algunos mss., en Mat_23:4 : «difíciles de llevar»; muchos mss. antiguos omiten este último pasaje. Véanse LLEVAR, ( NO ) PODER.¶
