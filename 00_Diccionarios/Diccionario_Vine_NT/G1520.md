# [[G1520]]

jeis (åἱ̂ò, [[G1520]]), uno. Se traduce ocasionalmente como «otro» cuando se expresa el segundo de un par (Mat_17:4; Mat_20:21; Mat_24:40-41; Mat_27:38; Mar_9:5; Jua_20:12; Gál_4:22; 1Ts_5:11). Véase UNO, etc.
