# VIANDA

broma (âñù̂ìá, [[G1033]]), para lo cual véase ALIMENTO, Nº 1, se traduce «vianda» en 1Co_3:2; 1Co_6:13, dos veces; 1Co_8:8; Heb_13:9. Véase también COMER, B, Nº 2.

Nota: Brosis, que significa el acto de comer (relacionado con bibrosko, comer), se traduce «viandas» en 1Co_8:4; véanse COMER, B, Nº 3, ORÍN.
