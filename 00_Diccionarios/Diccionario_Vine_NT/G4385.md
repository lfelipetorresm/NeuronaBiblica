# [[G4385]]

proteino (προτείνω, [[G4385]]), lit: estirar (pro, a; teino, estirar). Se usa en Hch_22:25, «le ataron» (vm: «extendido»), en referencia a los preparativos hechos para aplicar los azotes, probablemente extendiendo el cuerpo hacia adelante, para ponerlo tenso para que el castigo fuera más severo.¶
