# [[G3890]]

paramuthion (παραμύθιον, [[G3890]]), tiene e1 mismo significado que paramuthia (véase CONSOLAR, B, Nº 2), siendo 1a diferencia que paramuthia destaca el proceso o progreso del acto, y paramuthion el instrumento utilizado por e1 agente (Fil_2:1).¶

paramuthion (παραμύθιον, [[G3890]]), aliento. Se traduce: «refrigerio de amor» (Fil_2:1, (rv; Besson: «alivio»; rvr, rvr77, vm, lba: «consuelo»). Para consideraciones adicionales sobre este término, véase CONSUELO, Nº 2.¶
