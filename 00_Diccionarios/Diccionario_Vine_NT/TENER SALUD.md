# TENER SALUD

jugiaino (ὑãéáßíù, [[G5198]]), denota estar saludable, sano, tener salud (cf. el término castellano higiene). Se traduce «que tengas salud» ( 3Jn_1:2). Véanse SALUD, SALVO, SANO.
