# PROCLAMAR

keruáo (κηρύσσω, [[G2784]]), se traduce con el verbo proclamar en Mat_10:27 : «proclamadlo» (rv: «predicadlo»); Luc_12:3 : «se proclamará» (rv: «será pregonado»); véase PREDICAR, A, Nº 2, y también DIVULGAR, HERALDO ( SER ), PREGONAR, PUBLICAR.
