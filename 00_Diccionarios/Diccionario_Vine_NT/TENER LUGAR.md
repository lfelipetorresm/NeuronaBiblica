# TENER LUGAR

eukaireo (åὐêáéñÝù, [[G2119]]), tener oportunidad o tiempo libre, o dedicar ello a algo. Se traduce «ni aun tenían lugar de comer» (: «ni aun tenían tiempo para comer»). Véanse TENER OPORTUNIDAD, TENER TIEMPO, y, especialmente, INTERESAR bajo INTERéS, B, Nº 1. Cf. eukairia, véase OPORTUNIDAD, A, Nº 2.
