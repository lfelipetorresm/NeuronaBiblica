# ORILLA

Notas: (1) Para aigialos, véase PLAYA; (2) para queilos, véase LABIO; (3) para peran, «al otro lado», traducido «a la otra orilla» en Mat_8:28, véase LADO; (4) para el verbo prosormizo, traducido en Mar_6:53 : «arribaron a la orilla» (rv: «tomaron puerto»; rvr77: «atracaron»), véase ARRIBAR, Nº 6.¶
