# COMUNICAR

1. metadidomi (μεταδίδωμι, [[G3330]]), dar una parte de, impartir. Se traduce «comunicar(os)» en Rom_1:11 (rv: «repartir con»). Véase COMPARTIR.¶

2. prosanatithemi (προσανατίθημι, [[G4323]]), lit., poner delante (pros, hacia; ana, arriba, y tithemi, poner), esto es, poner un asunto ante otros para poder conseguir consejo o instrucción. Se usa de Pablo absteniéndose de consultar con nadie (Gál_1:16 : «consulté», rv: «conferí»; en 2:6: «me comunicaron», rv: «me dieron»). Cf. la forma más corta anatithemi, en 2:2: «expuse», usándose ahí la forma menos intensiva simplemente para significar la entrega de información, no el conferenciar con otros para conseguir consejo. Véase CONSULTAR.¶
