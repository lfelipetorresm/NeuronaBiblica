# VARÓN

1. aner (ἀíÞñ, [[G435]]), significa específicamente varón, no empleándose nunca del sexo femenino. Se traduce «varón» en pasajes como Mar_6:20; Luc_1:27, Luc_1:34; Luc_8:41; Luc_9:30, Luc_9:32; Luc_19:2; Luc_23:50, etc. Para un tratamiento detallado de este término, véase HOMBRE, A, Nº 2; véanse también ESPOSO, MARIDO.

2. arsen o arren (ἄñóçí, [[G730]]), se traduce «hombres» en Rom_1:27, tres veces; con el término «varón» en Mat_19:4; Mar_10:6; Luc_2:23; Gál_3:28, donde el significado es que en Cristo no hay distinción de sexos; el sexo no constituye una barrera ni para la salvación ni para el desarrollo de las gracias cristianas; Apo_12:5, Apo_12:13.¶

Nota: Para arsenokoites, «los que se echan con varones» (1Co_6:9), véase SODOMITA.
