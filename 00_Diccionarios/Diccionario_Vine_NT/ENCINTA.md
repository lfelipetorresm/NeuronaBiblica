# ENCINTA

A. Adjetivo

enkuos (ἔãêõïò, [[G1471]]), denota «encinta» (en, en, y kuo, concebir), (Luc_2:5).¶

B. Verbos

eco (ἔ÷ù, [[G2192]]), tener. Se usa en la expresión en gastri equein, que se traduce con el verbo concebir (Mat_1:18, Mat_1:23); y «estar encinta» (Mat_24:19; Mar_13:17; Luc_21:23; 1Ts_5:3; Apo_12:2). Véase CONCEBIR; véanse también AGONIZAR, ( TENER ) DOLENCIA, ENFERMEDAD, ( ESTAR ) ENFERMO, TENER ( DOLENCIA, ENFERMEDAD); véanse también CONTENER, TENER.
