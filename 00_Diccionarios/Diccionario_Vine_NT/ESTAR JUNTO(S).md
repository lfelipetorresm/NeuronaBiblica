# ESTAR JUNTO(S)

1. sunercomai (συνέρχομαι, [[G4905]]), véase REUNIR(SE). Se traduce «han estado juntos» en Hch_1:21.

2. sunomoreo (συνομορέω, [[G4927]]), bordear. Se usa de una casa contigua a una sinagoga, en Hch_18:7 : «junto» (Besson: «contigua a»; nvi: «lindante puerta con puerta»).¶
