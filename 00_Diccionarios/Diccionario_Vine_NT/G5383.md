# [[G5383]]

filoproteuo (φιλοπρωτεύω, [[G5383]]), lit., amar ser preeminente (filos, amante; proteuo, tener la preeminencia), codiciar tener el primer lugar. Se dice de Diótrefes ( 3Jn_1:9 : «al cual le gusta tener el primer lugar» (rvr; rv: «que ama tener el primado»).¶ 

filoproteuo (φιλοπρωτεύω, [[G5383]]), lit., amar ser preeminente (filos, amador), luchar por ser el primero. Se dice de Diótrefes ( 3Jn_1:9 : «Diótrefes, al cual le gusta tener el primer lugar»).¶ 

filoproteuo (φιλοπρωτεύω, [[G5383]]), véanse PREEMINENCIA, PRIMER LUGAR.
