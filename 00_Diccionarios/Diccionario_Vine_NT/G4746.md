# [[G4746]]

stoibas o stibas (óôïéâá́ò [[G4746]]), de steibo, pisar encima, denotaba primariamente una capa de hojas, cañas, ramitas o paja, sirviendo como cama; luego, una rama llena de hojas, de follaje suave como del que pudiera servirse uno para hacer una cama, o para pisar; «ramas» (Mar_11:8; RV: «hojas»).
