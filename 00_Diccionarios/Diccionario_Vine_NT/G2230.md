# [[G2230]]

jegemoneuo (ἡãåìïíåýù, [[G2230]]), ser un jegemon, conducir, ir adelante en el camino; vino a significar ser gobernador de una provincia. Se usa de Cirenio, gobernador de Siria (Luc_2:2; para las circunstancias, véase bajo CENSO); de Poncio Pilato, gobernador de Judea (Luc_3:1).¶ En la primera cláusula de este versículo el nombre jegemonia, gobierno o soberanía, se traduce «imperio»; Cf. el término castellano «hegemonía».¶
