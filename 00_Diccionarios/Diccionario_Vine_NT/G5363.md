# [[G5363]]

filanthropia (φιλανθρωπία, [[G5363]]), denota, lit: amor por el hombre (fileo, amar, querer, y anthropos, hombre); de ahí, bondad (Hch_28:2); en Tit_3:4 : «su amor para con los hombres».¶ Cf. el adverbio filanthropos, humanamente, bondadosamente.¶

Nota: (1) Juper (uper), por, tocante a, se traduce «por amor de» en 2Co_12:10, 2Co_12:15. Véanse ( EN ) CUANTO, ( POR ) CAUSA, FAVOR, MAS. (2) Para filarguria, amor al dinero (1Ti_6:10), véase DINERO (amor al). (3) Para filadelfia, véanse AFECTO, Nº 4, FRATERNAL.
