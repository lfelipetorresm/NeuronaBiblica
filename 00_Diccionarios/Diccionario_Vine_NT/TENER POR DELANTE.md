# TENER POR DELANTE

prokeimai (ðñüêåéìáé, [[G4295]]), lit: yacer delante (pro, delante; keimai, yacer), véase DELANTE, B, Nº 4. Se traduce «que tenemos por delante» (Heb_12:1, : «que nos es propuesta»). Véanse también DISPONER, PONER, PRIMERO, PROPONER.
