# [[G4829]]

summerizo (óõììåñßæù, [[G4829]]), primariamente, distribuir en partes (sun, con; meros, parte), en la voz media, tener una parte en. Se utiliza en 1Co_9:13 : «del altar participan» (vm: «participan juntamente con el altar»), esto es, se alimentan con otros de aquello que, habiendo sido sacrificado, ha sido puesto sobre un altar. De esta manera el creyente participa de Cristo, que es el altar mencionado en Heb_13:10.¶

Notas: (1) Para koinonia, véanse y; se traduce «participar» en 2Co_8:4; (2) para metalempsis, traducido «participasen» en 1Ti_4:3, véase B, Nº 3.¶
