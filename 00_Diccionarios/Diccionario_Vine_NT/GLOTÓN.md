# GLOTÓN

gaster (γαστήρ, [[G1064]]), denota vientre. Se usa en Tit_1:12, con el adjetivo argos, ocioso, metafóricamente, para significar un glotón: «glotones» (: «vientres»; vm: «glotones»); véase VIENTRE.
