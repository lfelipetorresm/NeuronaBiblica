# TENER SEÑORÍO

katakurieuo (κατακυριεύω, [[G2634]]), ejercer señorío (kata, abajo; kurios, señor), véase, B. Se traduce «teniendo señorío» (1Pe_5:3); véanse también DOMINAR, A, Nº 3, ENSEÑOREARSE, Nº 2.
