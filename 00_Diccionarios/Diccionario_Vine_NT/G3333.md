# [[G3333]]

metakaleo (μετακαλέω, [[G3333]]), (meta, implicando cambio, y Nº 1), llamar de un lugar a otro, hacer venir (cf. Ose_11:1, lxx). Se usa en la voz media tan solo, llamar para uno mismo, enviar a buscar, llamar aquí (Hch_7:14 : «hizo venir»; Hch_10:32 : «haz venir»; Hch_20:17 : «hizo llamar»; Hch_24:25 : «te llamaré»). Véase VENIR.¶
