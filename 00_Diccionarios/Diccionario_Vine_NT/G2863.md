# [[G2863]]

komao (κομάω, [[G2863]]), significa dejar crecer el cabello, llevar el cabello largo, una gloria para la mujer, una deshonra para el hombre, tal como lo enseña la naturaleza, (1Co_11:14-15). Véanse CRECER, DEJAR(SE).¶
