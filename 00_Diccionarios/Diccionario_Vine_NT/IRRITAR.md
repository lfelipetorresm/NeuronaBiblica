# IRRITAR

1. paroxuno (παροξύνω, [[G3947]]), primariamente, afilar (relacionado con paroxusmos ; véase DESACUERDO). Se usa metafóricamente, significando irritar, provocar, en la voz pasiva (Hch_17:16 : «su espíritu se enardecía»; 1Co_13:5 : «no se irrita»).¶

2. prokaleo (προκαλέω, [[G4292]]), llamar afuera, como para una lucha; de ahí, agitar lo que es malo en otro. Se usa en voz media en Gál_5:26 : «irritándonos unos a otros».¶
