# PRIMADO

Notas: (1) Para proteuo, traducido «el primado» en Col_1:18 (rv), referido a la preeminencia de Cristo «en todo», véase PREEMINENCIA, Nº 1;¶ (2) para filoproteuo, traducido «que ama tener el primado» ( 3Jn_1:9, rv), véase PREEMINENCIA, Nº 2; véanse asimismo bajo LUGAR y PRIMER LUGAR.¶
