# VIHUELA

Notas: (1) Para el nombre kithara, traducido «vihuela» en 1Co_14:7 (rv; rvr: «citara»), véase ARPA. (2) Para el verbo kitharizo, tocar o tañer el arpa, traducido «que se tañe … con la vihuela» (1Co_14:7, rv; rvr: «que se toca … con la cítara»), véase CÍTARA, B.
