# [[G2575]]

kaminos (κάμινος, [[G2575]]), horno (de donde viene el término latino caminus; cf. el castellano chimenea), usado para fundir, o para cocer cerámica. Se usa en Mat_13:42, Mat_13:50; Apo_1:15; Apo_9:2.¶
