# [[G1350]]

diktuon (δίκτυον, [[G1350]]), término general para una red (procedente de un antiguo verbo, diko, arrojar: relacionado con diskos, disco). Se utiliza en Mat_4:20-21; Mar_1:18-19; Luc_5:2, Luc_5:4-6; Jua_21:6, Jua_21:8, Jua_21:11, dos veces.¶ En la lxx se utiliza de una red para cazar aves (Pro_1:17); también se utiliza en otras formas, p.ej., en sentido figurado, de una trampa (Job_18:8; Pro_29:5).
