# [[G4979]]

scoinion (σχοινίον, [[G4979]]), diminutivo de scoinos, junco. Se usa de las cuerdas con las que Cristo hizo un látigo (Jua_2:15); de las amarras de un barco (Hch_27:32). Véase AMARRA.¶ 

scoinion (σχοινίον, [[G4979]]), amarra o cuerda, diminutivo de scoinos, junco. Significaba una cuerda hecha de juncos; denota: (a) una cuerda pequeña (Jua_2:15, plural), (b) una cuerda (amarras, Hch_27:32). Véase CUERDA.¶
