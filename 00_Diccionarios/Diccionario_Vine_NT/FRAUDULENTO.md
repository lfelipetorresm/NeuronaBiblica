# FRAUDULENTO

dolios (δόλιος, [[G1386]]), engañoso. Se usa en 2Co_11:13, de falsos apóstoles como «obreros fraudulentos» (rv, rvr, rvr77; Besson: «engañosos»; nm: «dolosos»). Cf. dolos ; véase ENGAÑAR, C, Nº 2, etc.
