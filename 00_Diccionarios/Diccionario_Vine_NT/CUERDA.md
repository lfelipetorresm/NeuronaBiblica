# CUERDA

scoinion (ó÷ïéíßïí, [[G4979]]), diminutivo de scoinos, junco. Se usa de las cuerdas con las que Cristo hizo un látigo (Jua_2:15); de las amarras de un barco (Hch_27:32). Véase AMARRA.¶
