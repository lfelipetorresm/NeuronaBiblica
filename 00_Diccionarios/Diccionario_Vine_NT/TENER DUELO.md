# TENER DUELO

Nota: Para pentheo, traducido «y no más bien tuvisteis duelo» (1Co_5:2, : «haberos lamentado»), véase LAMENTAR, A, Nº 3, y también TENER LUTO, TRISTE ( ESTAR ).
