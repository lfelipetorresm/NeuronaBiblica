# [[G247]]

1. alos (ἄëëùò, [[G247]]), adverbio que se corresponde con el adjetivo allos (véase OTRO), se traduce «de otra manera» en 1Ti_5:25. El contraste no es con obras que no son buenas, para lo cual se debería usar el adverbio jeteros, «de distinta manera», «de otro modo», cf. Fil_3:15, sino con buenas obras que no son evidentes.¶ Para este contraste, véase la diferencia correspondiente entre los adjetivos alos y jeteros bajo OTRO.
