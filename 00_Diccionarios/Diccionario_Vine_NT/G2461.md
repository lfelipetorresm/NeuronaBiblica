# [[G2461]]

jippikos (ἱððéêüò, [[G2461]]), adjetivo que significa «de un caballo» o «de jinetes», ecuestre. Se usa como nombre, denotando caballería (Apo_9:16 : «jinetes»), en una cantidad de «doscientos millones».¶
