# ENRAMADA

skene (σκηνή, [[G4633]]), tienda, cabaña, tabernáculo. Se usa de tiendas como moradas (Mat_17:4; Mar_9:5; Luc_9:33, traducido «enramadas»; Heb_11:9 : «tiendas»). Para un tratamiento completo, véase .
