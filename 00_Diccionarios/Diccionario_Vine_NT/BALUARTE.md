# BALUARTE

edraioma (ἐäñáßùìá, [[G1477]]), apoyo, baluarte, sujeción (de edraios, firme; de edra, asiento). Se traduce «baluarte» en 1Ti_3:15 (rvr, rvr77; rv, vm: «apoyo»). Se utiliza de una iglesia local.¶
