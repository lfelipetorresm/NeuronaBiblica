# [[G4323]]

prosanatithemi (ðñïóáíáôßèçìé, [[G4323]]), se traduce «consulté» en Gál_1:16. Véase COMUNICAR, Nº 2.

prosanatithemi (ðñïóáíáôßèçìé, [[G4323]]), lit., poner delante (pros, hacia; ana, arriba, y tithemi, poner), esto es, poner un asunto ante otros para poder conseguir consejo o instrucción. Se usa de Pablo absteniéndose de consultar con nadie (Gál_1:16 : «consulté», rv: «conferí»; en 2:6: «me comunicaron», rv: «me dieron»). Cf. la forma más corta anatithemi, en 2:2: «expuse», usándose ahí la forma menos intensiva simplemente para significar la entrega de información, no el conferenciar con otros para conseguir consejo. Véase CONSULTAR.¶
