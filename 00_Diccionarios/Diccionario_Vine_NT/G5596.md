# [[G5596]]

psomion (ψωμίον, [[G5596]]), diminutivo de psomos, bocado. Denota un fragmento, un bocado (relacionado con psomizo ; véase COMER) (Jua_13:26, dos veces, 27,30). Era una porción delgada de pan que se usaba como cuchara para tomar comida del plato común. Cf. Rut_2:14. Véase PAN.¶
