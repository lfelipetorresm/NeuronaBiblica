# TENER QUIETUD

jesucazo (ἡóõ÷Üæù, [[G2270]]), véase DESISTIR, etc. Se traduce «tener quietud» (1Ts_4:11, : «tener tranquilidad»). Cf. jesuquia, véanse TRANQUILIDAD y jesuquios, véanse REPOSADAMENTE, SOSEGADO.
