# [[G3046]]

litra (λίτρα, [[G3046]]), era una moneda siciliana, equivalente a una libra latina, o as (de litra proviene el nombre de la unidad métrica «litro»). En el NT se usa como medida de peso, una libra (Jua_12:3 : «una libra de perfume»; Jua_19:39 : «cien libras»).¶
