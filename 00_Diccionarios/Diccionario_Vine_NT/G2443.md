# [[G2443]]

jina (ἵíá, [[G2443]]), significa «para que»; en Apo_13:13 se traduce «de tal manera que» (Besson: «para que»); véase PARA QUE.

jina (ἵíá, [[G2443]]), a fin de que, para que. Se utiliza en multitud de pasajes, traducido generalmente «para que» (p.ej., Mat_1:22; Mat_2:15; Mat_4:14).
