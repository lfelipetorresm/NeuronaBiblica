# CONTESTAR

eipon (åἴðïí, [[G2036]]), que se usa como tiempo aoristo de lego, se traduce con el verbo contestar en Mat_21:24. Véanse AÑADIR, DECIR, HABLAR, LLAMAR, MANDAR, ORDENAR, REFERIR, RESPONDER.
