# [[G1159]]

dapanao (δαπανάω, [[G1159]]), denota: (a) gastar (Mar_5:26; Hch_21:24 : «paga sus gastos»; 2Co_12:15 a: «gastaré»; para «me gastaré del todo», véase Nº 2); (b) consumir, malgastar (Luc_15:14 : «malgastado»; Stg_4:3 : «gastar»). Véanse MALGASTAR, PAGAR.¶
