# ESTAR FUERA DE SÍ

existemi (ἐîßóôçìé, [[G1839]]), significa primaria y literalmente poner fuera de posición, desplazar; de ahí, significa estar fuera de sí (Mar_3:21; 2Co_5:13); en este último pasaje se contrasta con sofroneo, estar cuerdo. Véanse ASOMBRAR(SE), ( ESTAR ), ( ESTAR ) LOCO, MARAVILLAR(SE).
