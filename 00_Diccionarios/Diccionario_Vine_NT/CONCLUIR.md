# CONCLUIR

1. epiteleo (ἐðéôåëÝù, [[G2005]]), (epi, arriba, intensivo, y teleo, acabar), llevar a cabo; es una forma intensificada de este último verbo. El sentido es el de llevar a cabo a la perfección. Se traduce con el verbo concluir en Rom_15:28 : «haya concluido». Véanse ACABAR, CABO, CONCLUIR, CUMPLIR, ERIGIR, HACER, LLEVAR ( A CABO ), PERFECCIONAR.

2. logizomai (λογίζομαι, [[G3049]]), contar. Se traduce «concluimos» en Rom_3:28 (rv, rvr). Véase CONTAR, etc.
