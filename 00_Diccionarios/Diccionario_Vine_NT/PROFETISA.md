# PROFETISA

profetis (ðñïöç̂ôéò, [[G4398]]), forma femenina de profetes (véase PROFETA, Nº 1). Se utiliza de Ana, Luc_2:36; es el título que se autoatribuye «esa mujer Jezabel» en Apo_2:20.¶
