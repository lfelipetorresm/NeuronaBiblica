# TENER RESPETO

entrepo (ἐíôñÝðù, [[G1788]]), girar alrededor (en, en; trepo, girar). Se traduce como tener respeto en Mat_21:37; Mar_12:6; Luc_18:4; Luc_20:13; véanse AVERGONZAR, A, Nº 4, RESPETAR, REVERENCIA; cf. entrope, véase VERGÜENZA bajo AVERGONZAR, B, Nº 3.
