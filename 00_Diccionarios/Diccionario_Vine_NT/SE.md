# SE

jeautou (ἑáõôïõ̂, [[G1438]]), uno mismo, él mismo, reflexivo de autos (véase éL, Nº 1). Se traduce SE en pasajes como Mat_23:12, dos veces; Mar_6:51; etc. Véanse MISMO, Nº 5, PROPIO, Nº 2, etc.
