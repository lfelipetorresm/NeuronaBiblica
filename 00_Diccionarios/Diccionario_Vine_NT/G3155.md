# [[G3155]]

maten (μάτην, [[G3155]]), propiamente caso acusativo de mate, falta, insensatez. Significa en vano, sin objeto (Mat_15:9 : «en vano»; Mar_7:7 : «en vano»).¶

Nota: Para afron, traducido «hombres vanos» en 1Pe_2:15 (rv; rvr: «insensatos»), véase INSENSATO bajo INSENSATEZ, B, Nº 3.
