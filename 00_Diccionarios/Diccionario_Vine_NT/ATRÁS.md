# ATRÁS

opiso (ὀñßóù, [[G3694]]), relacionado con epomai, seguir. Se utiliza adverbialmente, de lugar, con el significado «atrás», en la frase eis ta opiso ; lit: hacia las cosas atrás (Mar_13:16; Luc_9:62; Luc_17:31; Jua_6:66; Jua_18:6; Jua_20:14. Cf. Fil_3:13 : «lo que queda atrás»). Véase TRAS.¶
