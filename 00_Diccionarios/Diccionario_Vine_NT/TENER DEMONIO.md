# TENER DEMONIO

daimonizomai (δαιμονίζομαι, [[G1139]]), estar poseído por un demonio o por demonios. Se traduce «que había tenido el demonio» (Mar_5:16); véase DEMONIO, C; también ATORMENTAR, A, Nº 2.
