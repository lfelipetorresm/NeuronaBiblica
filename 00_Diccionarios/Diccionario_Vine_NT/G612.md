# [[G612]]

apokrisis (ἀðüêñéóéò, [[G612]]), lit. una separación o distinción, es el término regularmente utilizado para denotar «respuesta» (Luc_2:47; Luc_20:26; Jua_1:22 y Jua_19:9).¶
