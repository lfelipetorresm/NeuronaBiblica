# RUGIR

1. mukaomai (μυκάομαι, [[G3455]]), utilizado propiamente de los bueyes, es un término onomatopéyico, mugir, bramar. Se utiliza de un león (Apo_10:3 : «ruge», rv, rvr, rvr77, vm, lba; cf. nvi, que traduce «rugido»).¶

2. oruomai (ὠñýïìáé, [[G5612]]), aullar o rugir, onomatopéyico, de animales u hombres. Se utiliza de un león (1Pe_5:8), como símil de Satanás: «rugiente», lit. «que ruge».¶
