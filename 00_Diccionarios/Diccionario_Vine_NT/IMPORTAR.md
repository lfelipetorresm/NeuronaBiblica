# IMPORTAR

1. diafero (διαφέρω, [[G1308]]), se traduce con el verbo importar, en el sentido de tener importancia para uno, con ouden, nada, utilizado adverbialmente, esto es, «no tiene para mí ninguna importancia»; la comisión recibida directamente del Señor liberaba a Pablo de tener responsabilidad frente a la autoridad de los apóstoles. Véanse ATRAVESAR, DIFERENTE, DIFERIR, DIFUNDIR, LLEVAR, MEJOR, TRAVéS, VALER.

2. melei (μέλει, [[G3199]]), se traduce con el verbo importar en Jua_10:13 : «no le importan» (vha: «no se le da cuidado»). Véanse CUIDAR, A, Nº 1, DAR, Nº 23, DAR CUIDADO, etc.
