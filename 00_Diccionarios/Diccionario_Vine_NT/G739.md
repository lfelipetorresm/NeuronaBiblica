# [[G739]]

artios (ἄñôéïò, [[G739]]), ajustado, completo (de artos, miembro, articulación). Se utiliza en 2Ti_3:17 : «perfecto».¶

Notas: (1) pleroo, llenar, completar, cumplir, se traduce «perfectas» en Apo_3:2, de las obras (Besson: «cumplidas»); véanse CUMPLIR, A, Nº 5, LLENAR, A, Nº 1; (2) teleioo, véase A, Nº 1, se traduce «sea perfecto» (Fil_3:12); «hecho perfecto» (Heb_7:28); «hacer perfecto» (Heb_9:9); «hacer perfectos» (Heb_10:1); «hizo perfectos» (v. 14); «hechos perfectos» (Heb_12:23); véanse también ACABAR, Nº 6, CUMPLIR, A, Nº 14; (3) para teleiotes, traducido «vínculo perfecto» en la rvr en Col_3:14, véase PERFECCIÓN, Nº 4.
