# [[G5108]]

toioutos (ôïéïõ̂ôïò, [[G5108]]), significa «tal», y se traduce «cosa semejante» en Efe_5:27; «semejante» (Stg_4:16). Véanse TAL, y también HOMBRE, PERSONA. 

toitoutos (ôïéôïõ̂ôïò, [[G5108]]), adjetivo que significa «tal», se usa frecuentemente como nombre (p.ej., Rom_16:18), traducido «tales personas», o «tal hombre» (2Co_12:5).

Nota: El adjetivo oligopistos, lit., «de poca fe», se traduce invariablemente «hombre/s de poca fe» (Mat_6:30; Mat_8:26; Mat_14:31; Mat_16:8; Luc_12:28); véase FE, B.¶
