# DISTRIBUCIÓN

diakonia (äéáêïíßá, [[G1248]]), el oficio y obra de un diakonos (véase MINISTRO). Se traduce «distribución» en Hch_6:1 (rvr, rvr77; rv: «ministerio»). Véase MINISTERIO, etc.
