# RETROCEDER

Notas: (1) Para jupostelo, traducido «si retrocediere» (Heb_10:38; rv: «se retirare»), véase RETRAER;

(2) para jupostole (Heb_10:39 : «de los que retroceden»; rv: «que nos retiremos»), véase RETIRAR, Nota bajo Nº 4;

(3) en Jua_18:6 : «retrocedieron» (rvr), es, lit. «se fueron (apercomai) para atrás» (cf. rv: «volvieron atrás»). Véanse IR, MARCHARSE, SALIR.
