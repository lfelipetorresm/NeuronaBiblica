# PENA

1. dike (δίκη, [[G1349]]), justicia, o la ejecución de una sentencia. Se traduce «sufrirán pena» en 2Ts_1:9 (rv: «serán castigados»). Véase JUSTICIA, Nº 1, etc.

Nota: Para molis y mogis, traducidos «a duras penas», véase DURO bajo B, Nº 3 y 4.
