# IMPEDIR

1. enkopto (ἐãêüðôù, [[G1465]]), véanse ESTORBAR, ESTORBO, A, Nº 1. Se traduce «me he visto impedido» en Rom_15:22.

2. koluo (êùëýù, [[G2967]]), impedir, prohibir, refrenar. Se traduce impedir en Mat_9:14 : «no se lo impidáis»; Mar_10:14 : igual; Luc_11:52 : «se lo impedisteis»; Luc_18:16 : «no se lo impidáis»; Hch_8:36 : «¿qué impide … ?»; Hch_10:47 : «impedir»; Hch_24:23 : «impidiese»; Hch_27:43 : «les impidió»; 1Co_14:39 : «no impidáis»; 1Ts_2:16 : «impidiéndonos»; véase PROHIBIR, etc.

3. katapauo (êáôáðáýù, [[G2664]]), véase DAR REPOSO.

4. sfragizo (óöñáãßæù, [[G4972]]), véase SELLAR. Se traduce «no se me impedirá este mi gloria» en 2Co_11:10 (tr); en los mss. más comúnmente aceptados aparece el verbo fraáo, cerrar, tapar; véase CERAR, Nº 4.
