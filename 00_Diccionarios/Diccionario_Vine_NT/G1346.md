# [[G1346]]

dikaios (διακαίως, [[G1346]]), justamente, rectamente, de acuerdo con lo que es recto. Se dice: (a) del juicio de Dios (1Pe_2:23 : «que juzga justamente»); (b) de hombres (Luc_23:41 : «justamente»; 1Co_15:34 : «debidamente», vm: «como es justo»; 1Ts_2:10 : «justa … mente»; Tit_2:12 : «justa … mente»). Véase DEBIDAMENTE.¶
