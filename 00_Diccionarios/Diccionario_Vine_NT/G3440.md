# [[G3440]]

monos (μόνος, [[G3440]]), denota solo, solitario, y se traduce «solo», p.ej., en Mat_14:23; Mat_17:8; Mat_18:15, etc. Se utiliza como atributo de Dios: «único» (Jua_5:44; Jua_17:3; Rom_16:27; 1Ti_1:17); 1Ti_6:15 : «solo»; v. 16: «único»; Jud_1:4,25: «único»; Apo_15:4 : «solo». Véanse APARTE.
