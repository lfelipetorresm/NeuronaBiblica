# INJURIADOR, INJURIOSO, INJURIAR

A. Nombre

jubristes (ὑâñéóôÞò, [[G5197]]), hombre violento, insolente (relacionado con jubris (véanse AFRENTA, PERJUICIO). Se traduce «injuriosos» (Rom_1:30); «injuriador» (1Ti_1:13).¶

2. loidoreo (ëïéäïñÝù, [[G3058]]), denota agraviar, insultar (Jua_9:28 : «injuriaron»; Hch_23:4 : «injurias»; 1Co_4:12 : «nos maldicen»; 1Pe_2:23 : «le maldecían»). Véase MALDECIR.¶
