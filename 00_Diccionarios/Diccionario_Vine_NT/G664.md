# [[G664]]

apotomos (ἀðïôüìùò, [[G664]]), significa abruptamente; lit: de una manera cortante [apo, de (partitivo); temno, cortar], y por ello aceradamente, severamente (2Co_13:10 : «para no usar de severidad»; rv: «con dureza»); Tit_1:13, de reprender: «duramente»; véase SEVERIDAD.¶ Cf. apotomia, severidad.
