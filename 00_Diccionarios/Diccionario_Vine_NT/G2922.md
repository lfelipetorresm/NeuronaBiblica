# [[G2922]]

kriterion (êñéôÞñéïí, [[G2922]]), significa primariamente un medio de enjuiciar (relacionado con krino, juzgar; cf. el término castellano criterio), luego, un tribunal, juzgado, o un pleito [1Co_6:2, última cláusula; véase JUICIO, B, Notas (1) ; 1Co_6:4, para lo cual véase JUICIO, A, Notas (1) ; Stg_2:6].¶
