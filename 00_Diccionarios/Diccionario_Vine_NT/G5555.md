# [[G5555]]

crusolithos (χρυσόλιθος, [[G5555]]), lit., piedra de oro (crusos, oro; lithos, piedra). Es el nombre de una piedra preciosa de un color de oro, que ahora recibe el nombre de topacio (Apo_21:20), y así es traducido en la vm .¶
