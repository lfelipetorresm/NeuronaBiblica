# PERTENECER

eco (ἔ÷ù, [[G2192]]), tener. Se traduce «que pertenecen», en Heb_6:9 : «cosas mejores, y que pertenecen a la salvación» (vm: «que acompañan»). Véase TENER y cf. VECINO.

Nota: Para el verbo meteco, «pertenecía a otra tribu» (Heb_7:13, vm), véase PARTICIPAR, A, Nº 4.
