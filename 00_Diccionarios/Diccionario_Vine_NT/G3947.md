# [[G3947]]

paroxuno (ðáñïîýíù, [[G3947]]), primariamente, afilar (relacionado con paroxusmos ; véase DESACUERDO). Se usa metafóricamente, significando irritar, provocar, en la voz pasiva (Hch_17:16 : «su espíritu se enardecía»; 1Co_13:5 : «no se irrita»).¶

paroxuno (ðáñïîýíù, [[G3947]]), «se enardecía» (en Hch_17:16; véase IRRITAR).
